## Site Tom Troc

## Pour utiliser ce projet : 

- Commencer par cloner le projet. 
- installez le projet chez vous, dans un dossier exécuté par un serveur local (type MAMP, WAMP, LAMP, etc...)
- Importez le fichier `site_tomtroc.sql` dans votre base de données.

## Lancez le projet ! 

Pour pouvoir utiliser le projet, il vous faudra modifier le fichier à l'emplacement suivant : `app/Core/Database.php`, et rajoute votre identifiant et mot de passe de connexion à la base de donnée
Des données sont déjà presentes pour avoir un peu de contenu, vous pouvez créer des comptes sur le site ( attention, vous ne pouvez pas utilisé de mail ou pseudo déja existant )
